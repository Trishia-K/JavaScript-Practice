# JavaScript Practice & Challenges 

A collection of JavaScript exercises and samll projects built 
for me to learn and perfect my JavaScript skills.

## Challenges Completed

| File | Description | Concepts Practiced |
|---|---|---|
| `Greeting.js` | Smart greeting function based on name and time of day 
| `Tax Calculator.js` | Calculates tax based on income bracket
| `Grade Calculator.js` | Takes exam scores and returns a grade classification 
| `Password validator.js` | Validates password strength and lists what is missing 

## Concepts Covered So Far
- Variables — `let`, `const`
- Functions and parameters
- Conditionals — `if`, `else if`, `else`
- Loops — `for`
- Arrays and array methods — `push`
- Template literals
- Regular expressions (Regex) — `/\d/`, `/[A-Z]/`, `/[!@#$%]/`
- DOM interaction — `prompt()`, `console.log()`
